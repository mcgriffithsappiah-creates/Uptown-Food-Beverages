# Uptown Food & Beverages Website

A modern landing page for Uptown Food & Beverages, built as a responsive static site.

## Included files
- `index.html` — main website content
- `styles.css` — responsive styling and layout
- `script.js` — mobile menu toggle behavior

## Preview locally
Open `index.html` in a browser, or use a local server such as:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes
- The location section includes a Google Maps embed for the store coordinates.
- The design is optimized for desktop and mobile screens.
